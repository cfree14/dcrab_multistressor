# Multistressor management strategies in the OR/WA Dungeness crab fishery

This is the GitHub repository for the MSE-subteam working on a proposal to understand the impact of multiple climate stressors in the OR/WA Dungeness crab fishery and to evaluate the performance of strategies that may be robust to these changes.
